# Persona 4 Golden Load Remover

# Download
https://github.com/hoXyy/LiveSplit.AutoSplitters.WASM/releases/download/latest/p4g_load_remover.wasm

# Build
1. Get Rust from https://rustup.rs.
2. Add wasm32 target with `rustup target wasm32-unknown-unknown`.
3. Build with `cargo build --release --package p4g-load-remover` from the repository root.
