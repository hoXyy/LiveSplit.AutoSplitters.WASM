# Saints Row 2 Autosplitter and Load Remover

# Download
https://github.com/hoXyy/LiveSplit.AutoSplitters.WASM/releases/download/latest/sr2_autosplitter.wasm

# Build
1. Get Rust from https://rustup.rs.
2. Add wasm32 target with `rustup target wasm32-unknown-unknown`.
3. Build with `cargo build --release --package sr2-autosplitter` from the repository root.
