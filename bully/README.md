# Bully Autosplitter

# Download
https://github.com/hoXyy/LiveSplit.AutoSplitters.WASM/releases/download/latest/bully_autosplitter.wasm

# Build
1. Get Rust from https://rustup.rs.
2. Add the wasm32 target with `rustup target add wasm32-unknown-unknown`.
3. Build with `cargo build --release --package bully-autosplitter` from the repository root.
