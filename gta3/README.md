# GTA III Autosplitter

## Download
https://github.com/hoXyy/LiveSplit.AutoSplitters.WASM/releases/download/latest/gta3_autosplitter.wasm

## Supported versions

- 1.0
- 1.1
- Steam
- Japanese (not tested)

## Build
1. Get Rust from https://rustup.rs.
2. Add wasm32 target with `rustup target wasm32-unknown-unknown`.
3. Build with `cargo build --release --package gta3-autosplitter` from the repository root.
