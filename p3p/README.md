# Persona 3 Portable Load Remover

# Download
https://github.com/hoXyy/LiveSplit.AutoSplitters.WASM/releases/download/latest/p3p_load_remover.wasm

# Build
1. Get Rust from https://rustup.rs.
2. Add wasm32 target with `rustup target wasm32-unknown-unknown`.
3. Build with `cargo build --release --package p3p-load-remover` from the repository root.
