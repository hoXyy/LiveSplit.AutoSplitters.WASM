# GTA IV Autosplitter and Load Remover

Only Patch 4 currently supported.

# Download
https://github.com/hoXyy/LiveSplit.AutoSplitters.WASM/releases/download/latest/gta4_autosplitter.wasm

# Build
1. Get Rust from https://rustup.rs.
2. Add wasm32 target with `rustup target wasm32-unknown-unknown`.
3. Build with `cargo build --release --package gta4-autosplitter` from the repository root.
